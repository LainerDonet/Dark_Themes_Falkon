Tek-Phase Dark - Falkon Browser Theme
======================================
A clean dark theme using pure blacks (#111111, #1C1C1C),
mid-grays (#333-#666) and white/light gray accents (#CCC-#EEE).

INSTALL:
  Copy the folder "tek-phase-dark/" to:
    ~/.local/share/falkon/themes/

  Then in Falkon: Preferences → Appearance → Theme → Tek-Phase Dark

NOTES:
  If the original theme had transp.png / semi-transp.png, copy those
  from Tek-Phase Blue into this theme's images/ folder.
  The Play font is recommended (included in most Linux distros).
